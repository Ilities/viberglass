   - Function signature: `useInterval(callback: () => void, delay: number | null)`

Create usePolling hook that extends useInterval with API fetching and visibility detection:

1. Create `usePolling.ts` with:
   - Generic interface UsePollingOptions<T> with:
     - fn: () => Promise<T> - async function to poll
     - interval: number - polling interval in milliseconds
     - immediate?: boolean - call immediately on mount (default true)
     - enabled?: boolean - enable/disable polling (default true)
     - onComplete?: (data: T) => boolean - return true to stop polling
   - State: data, error, isPolling, isPaused
   - Page Visibility API integration:
     - Listen to 'visibilitychange' event
     - Set isPaused = document.hidden
     - When tab becomes visible again, trigger immediate refetch
   - Poll function that:
     - Skips if isPaused or already loading
     - Calls fn() and updates data/error
     - Checks onComplete to stop polling
   - Import and use useInterval for timing

2. Export return object: { data, error, isPolling, isPaused, poll, refetch }

Do NOT poll when tab is hidden (wastes resources). Do NOT start new poll if isPolling is true (prevents overlapping requests).
