Create WebhookSecretService.ts.

**WebhookSecretService class:**
Constructor:
- constructor(private credentialProvider: CredentialProvider)

Methods:
- async getSecret(config: WebhookProviderConfig, projectId?: string): Promise<string>
  - Switch on config.secretLocation:
    - 'database': return config.webhookSecret (decrypted from webhook_secret_encrypted column)
    - 'ssm': Use credentialProvider.get(tenantId, config.secretPath)
    - 'env': return process.env[config.secretPath || 'WEBHOOK_SECRET']
  - Throw error if secret is empty/undefined

- async encryptSecret(secret: string): Promise<string>
  - For database storage, encrypt using AES-256-GCM
  - Use crypto.randomBytes(12) for IV
  - Return format: base64(iv):base64(ciphertext):base64(authTag)

- async decryptSecret(encrypted: string): Promise<string>
  - Parse IV, ciphertext, authTag from encrypted format
  - Decrypt using AES-256-GCM
  - Return plaintext secret

Note: For Phase 8 scope, the encryption can be a simple implementation. The key decision is WHERE to store (database vs SSM), not the encryption algorithm specifics (which can be refined).

Import CredentialProvider from existing credentials module.
</action>
