<objective>
Remove the unused S3+CloudFront frontend static hosting infrastructure from the Pulumi codebase, since Amplify SSR is now the deployment method.

Purpose: Reduce AWS costs (~$15-20/month for CloudFront), avoid confusion about deployment method, and simplify infrastructure maintenance.

Output: Clean Pulumi stack without frontend S3+CloudFront resources, updated documentation reflecting Amplify SSR deployment.
</objective>
